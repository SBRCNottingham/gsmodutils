gsmodutils.scrumpy.get_tokens(line_dt)[source]¶

Goes through each charachter in scrumpy file attempting to find tokens

BUG: if there is a numeric after a direction token this fails e.g. ‘->2 “PROTON”’ fails but ‘-> 2 “PROTON”’ works

gsmodutils.scrumpy.load_scrumpy_model(filepath, atpase_reaction='ATPase', atpase_flux=3.0, media=None, objective_reactions=None, obj_dir='max')[source]¶

Specify a base scrumpy structural model file and returns a cameo model. This hasn’t be thoroughly tested so expect there to be bugs

To get a solution from the returned object you need to specify nice stuff like the atpase reaction and media
